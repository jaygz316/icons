# Contributing to Jay's Icons

Thank you for contributing to this icon library.

## Guidelines

1.  **File Format**:
    *   **SVG**: Preferred for vector graphics.
    *   **PNG**: Minimum 256x256px resolution for raster images.
    *   **ICO**: For favicons only.

2.  **File Naming**:
    *   Use `camelCase` for consistency (e.g., `nationalGeneral.png`, `iptvBoss.png`).
    *   Avoid spaces or special characters.

3.  **Organization**:
    *   Place files in their respective folders: `svg/`, `png/`, `jpg/`, `ico/`, etc.
    *   If a new format is added, create a new folder for it.

4.  **Submission**:
    *   Create a new branch for your changes.
    *   Submit a Pull Request (PR) for review.
